# Events Subscription

A standard QRL-compatible node can expose subscription endpoints, and some nodes may provide additional custom subscriptions. As you can find in the [Supported Subscriptions](/guides/events_subscriptions/supported_subscriptions) guide, QRL Web3.js enables you to subscribe to supported events out of the box. It also provides the capability to subscribe to custom subscriptions as described in the [Custom Subscriptions](/guides/events_subscriptions/custom_subscriptions) guide.

:::important
If you provide custom subscriptions to users, consider building a QRL Web3.js plugin after you go through the [Custom Subscription](/guides/events_subscriptions/custom_subscriptions) guide. You can find how to develop a plugin at [QRL Web3.js Plugin Developer Guide](/guides/web3_plugin_guide/plugin_authors).
:::
