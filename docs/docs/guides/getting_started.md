# Getting Started

Welcome to the QRL Web3.js documentation.

QRL Web3.js is a collection of libraries that allow you to interact with a local or remote QRL-compatible node using HTTP, IPC, or WebSocket.
The following documentation covers common use cases and the API reference for the `@theqrl/*` package set.
